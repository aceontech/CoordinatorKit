# CoordinatorKit
A collection of Swift helpers to facilitate the Coordinator pattern on iOS.

## Installation

### Carthage

```
github "jarrroo/CoordinatorKit"
```

## Swift version compatibility

### Swift 3
CoordinatorKit 2.0 and up

### Swift 2.3
CoordinatorKit 1.1.1

### Swift 2.2
CoordinatorKit 1.1.0
